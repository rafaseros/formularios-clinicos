2. PROPÓSITO

Monitoreo gráfico y tabular de signos vitales durante la internación. Permite:

  • Registro gráfico de temperatura (35-41°C)
  • Registro gráfico de pulso (40-160 lpm)
  • Registro gráfico de presión arterial (60-200 mmHg)
  • Registro gráfico de frecuencia respiratoria (10-40 rpm)
  • Tabla de datos adicionales (peso, saturación O2, glucemia)

3. INSTRUCCIONES DE LLENADO

SECCIÓN A: IDENTIFICACIÓN

Nombres
El campo de nombres tiene espacio amplio para hasta 3 nombres. Ejemplo: Carlos Alberto José

SECCIÓN B: DATOS ANTROPOMÉTRICOS

Se registran UNA VEZ al inicio. No deben quedar espacios en blanco.

Talla
En centímetros. Ejemplo: 170 cm
Peso al ingreso
En kilogramos. Ejemplo: 72 kg
IMC
Índice de Masa Corporal (Peso/Talla²). Ejemplo: 24.9
ALERGIAS
Campo destacado en amarillo. Si no tiene, escribir "NINGUNA". Ejemplo: Penicilina

SECCIONES C-F: GRÁFICAS

Registrar los valores en las gráficas correspondientes marcando con un punto y uniendo con líneas. Turnos: M (Mañana), T (Tarde), N (Noche).

Temperatura
Escala 35-41°C. Registrar en cada turno.
Pulso
Escala 40-160 lpm. Registrar en cada turno.
Presión Arterial
Escala 60-200 mmHg. Registrar en cada turno.
Frecuencia Respiratoria
Escala 10-40 rpm. Registrar en cada turno.

SECCIÓN G: TABLA DE DATOS ADICIONALES

Registrar diariamente: peso, saturación O2 y glucemia (si aplica).

SECCIÓN I: RESPONSABLES POR TURNO

Registrar iniciales o nombre del responsable de cada turno.

5. ERRORES COMUNES Y CÓMO EVITARLOS

Error Consecuencia Prevención
No registrar alergias Riesgo para el paciente Siempre registrar en sección B
Registros incompletos en gráficas Información insuficiente Registrar todos los valores diariamente
Valores fuera de rango no marcados Dificulta identificación de problemas Marcar claramente valores anormales
No registrar peso diario Información incompleta Registrar peso cada día

6. CHECKLIST DE VERIFICACIÓN

  • Datos de identificación completos
  • Datos antropométricos registrados (talla, peso, IMC)
  • Alergias documentadas (o "NINGUNA")
  • Gráficas completas para cada día (temperatura, pulso, PA, FR)
  • Tabla de datos adicionales completada
  • Responsables por turno registrados
